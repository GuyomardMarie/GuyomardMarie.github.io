📍 Université de Pharmacologie d’Aix-Marseille

2024 & 2025

💻 Programming Languages and Coding
🎓 Master 2, Ingénierie de la Santé
This module is designed for students with a degree in pharmacology who are pursuing a master’s program focused on artificial intelligence. The first session introduces Python and the Pandas library for data management. Students learn how to prepare, explore, and visualize their data. Then, the module covers the basics of machine learning, including linear regression and support vector machines.



📍 Polytech Nice-Sophia (Université Côte d’Azur)

2020, 2021 & 2022

📊 Data Valorization
🎓 Master 1, parcours Mathématiques Appliquées et Modélisation
This module introduces fundamental concepts of statistical learning: maximum likelihood estimation, naïve Bayes test, logistic regression, and decision trees. Practical applications were implemented using RStudio.

💻 Introduction à la Programmation Algorithmique
🎓 Licence 3, parcours Mathématiques Appliquées et Modélisation
This module combined an introduction to Linux operating systems (permission management, administration, and software installation on Ubuntu) with an introduction to object-oriented programming (OOP) in Java. The topics covered included class creation (instances, constructors, methods), inheritance, error handling, package creation, and documentation using Doxygen.

🗃️ Big Data
🎓 Master 2, parcours Mathématiques Appliquées et Modélisation
This module aims to introduce tools for managing and analyzing large databases. Students were trained in the following technologies: storage using HDFS (Hadoop Distributed File System), resource management with YARN, and the use of frameworks such as Spark (MapReduce), HBase (NoSQL), and MongoDB.
